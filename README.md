Apple and Banana Wars | Unreal Engine Arena Shooter
Project Overview
Apple and Banana Wars is a high-octane, first-person arena shooter developed in Unreal Engine. The project demonstrates a "grit-meets-whimsy" aesthetic, featuring battle-hardened fruit combatants in a pressurized space-station environment. This project showcases the implementation of scalable AI systems, responsive player controls, and dynamic UI elements.

Technical Highlights
AI Architecture: Developed a robust AI system using Behavior Trees and Blackboards. Implemented AI Perception (Sight Sense) for reactive combat behaviors, including "Set Focus" and automated projectile firing.

System Inheritance: Utilized a Parent/Child Blueprint architecture to create a scalable character system, allowing for rapid iteration of enemy types (Apples vs. Bananas) while maintaining a centralized logic brain.

Animation & Meshes: Integrated Skeletal Meshes and custom Animation Blueprints to drive movement logic and state transitions.

UX/UI Design: Created a dual-layer interface including a persistent 2D player HUD and 3D Spatial Widgets (billboarding health bars) for real-time enemy status updates.

Gameplay Systems: Engineered a standardized ApplyDamage system, interactive physics-based pickups (Health/Speed boosts), and an automated respawn cycle.

Gameplay Experience
The core loop focuses on high-speed mobility and resource management. Players must navigate a complex 3D arena, utilizing cover to break AI line-of-sight while securing tactical power-ups to survive against increasing odds.
